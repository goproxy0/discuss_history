#5 dtma...@163.com 
password？根本没加密，只能算是一个防盗用识别串
无password参数：goagent = ['http://xxx.appspot.com/fetch1.php']
有password参数：goagent = [{'url':'http://xxx.appspot.com/fetch1.php', 'key':'xxxxx'}]
多个app：
goagent = [
'http://xxx1.appspot.com/fetch1.php',
'http://xxx2.appspot.com/fetch1.php',
{'url':'http://xxx3.appspot.com/fetch1.php', 'key':'xxxxx'},
{'url':'http://xxx4.appspot.com/fetch1.php', 'key':'xxxxx'},
]

默认使用gaeproxy服务端，如果需要使用goagent服务端，加入：
def find_http_handler(method, url, headers):
    return goagent

#10 dtma...@163.com 
1、WP内置6个共享服务端，解压后什么都不用修改直接运行WallProxy.exe即可。
2、如果需要使用自己的服务端，上传（可以使用SDUpload）后配置文件proxy.conf中以下部分
#gaeproxy = ['http://.appspot.com/fetch.php']
改为
gaeproxy = ['http://your-appid.appspot.com/fetch.php']
即可（去掉#再将url补充完整）
