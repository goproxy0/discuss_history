1.问题描述
在linux mint下，facebook的status post出现无限等待的情况。不论等待多久，post都没发出去。comment回复同理。虽然facebook上显示已经有comment，但是刷新网页后comment消失（说明实际上没法出去）。


2.系统信息及依赖情况
GoAgent Version    : 3.1.2 (python/2.7.4 gevent/1.0rc2 pyopenssl/0.13.1)
Listen Address     : 127.0.0.1:8087
Debug INFO         : 1
GAE Mode           : https
GAE Profile        : ipv4
gevent, greenlet, pyopenssl都正确安装（且版本最新）。用Mozilla Firefox for linux mint 24.0. Autoproxy 0.4b2.2013051811. certutil也配置过，证书导入没问题。Firefox证书导入也没问题。在Windows下使用没有出现该问题
