Issue 19326: 修改proxy.py，扩宽未知ip段，youtube提速
2 people starred this issue and may be notified of changes. Back to list
Status:  New
Owner:  ----
Type-Defect
Priority-Medium


 
Reported by 137...@gmail.com, Jan 4, 2015 

proxy.py中有一段google ip识别代码：

def is_google_ip(ipaddr):
    if ipaddr in ('74.125.127.102', '74.125.155.102', '74.125.39.102', '74.125.39.113', '209.85.229.138'):
        return False
    if ipaddr.startswith(('173.194.', '207.126.', '209.85.', '216.239.', '64.18.', '64.233.', '66.102.', '66.249.', '72.14.', '74.125.')):
        return True
    return False


貌似是黑名单与白名单，限制未知的 ip，导致可用 ip段的范围过窄；解除后有明显的提速，间接舒缓连接卡顿的问题。

修改第二段：
# if ipaddr.startswith(('173.194.', '207.126.', '209.85.', '216.239.', '64.18.', '64.233.', '66.102.', '66.249.', '72.14.', '74.125.')):
        # return True
    return True#False

解除白名单，放宽未知ip；
原本youtube速度<400KB/s，有卡顿；解除后的速度>400KB/s，卡顿减少。


Ps：
1.貌似是安全代码，不知有什么风险；
2.运行goagent后，开y2b视频能加快寻ip的速度，也可以让它自己慢慢寻。
