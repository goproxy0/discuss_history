#1 ericcl...@gmail.com 
具体原因不明，如果只是想要能用的话倒是简单。
你去local/proxy.py里把这三行前面各加上一个#字符
    if os.name == 'posix':
        import resource
        resource.setrlimit(resource.RLIMIT_NOFILE, (8192, -1))
对了，如果你的编辑器支持行号的话，行号是2264-2266
Apr 22, 2013
#2 eggcaker@gmail.com 
or, you can change the last line to this : 
resource.setrlimit(resource.RLIMIT_STACK, (resource.RLIM_INFINITY, resource.RLIM_INFINITY))

to set the stack size to unlimited

more info : 
http://stackoverflow.com/questions/10155301/python-valueerror-not-allowed-to-raise-maximum-limit
