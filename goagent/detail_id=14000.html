#4 tomb...@gmail.com 
据我所知nslookup可以得到指定国家的所有googleIP，但每个国家都不一样，必须用指定国家的IP才能得到那个国家所用的googleIP，比方说在大陆得到的googleIP就是美国IP，在台湾得到就是google台湾IP，在韩国得到的就是google韩国IP，所以想得到哪个国家所有的googleIP必须用那个国家的IP去执行nslookup指令才行，这个是难点。再说细点，实际上一个国家不同电信运营商得到的googleIP也不一样，比方说台湾中华电信得到就是中华电信的GGCIP，台湾远传得到就是远传的GGCIP，当然他们得先有GGC。

#8 zhanqyi...@gmail.com 
1/ 拿到一组 GGC ip 地址之后
例如 61.219.131.212 / 213 / 214 / 215 ...
首先取出一个
在浏览器里打开
看看能否进入谷歌页面（也就是 www.google.com 所显示的页面）

2/ 如果能进入谷歌页面
再打开“命令提示符”（也就是 DOS 窗口）
执行 ping 61.219.131.212 命令
看看速度快不快

3/ 如果速度较快
将它放在 proxy.ini 文件的 google_hk 后面
例如 google_hk = 61.219.131.212|61.219.131.213|61.219.131.214|61.219.131.2125
大致就是这样

#15 yangyang...@gmail.com 
“想得到哪个国家所有的googleIP必须用那个国家的IP去执行nslookup指令才行，这个是难点”，4 楼，可借由 tor 实现自定义出口 IP，比如，要指定台湾 IP，可在其配置文件 /usr/local/etc/tor/torrc 中增加 ExitNodes {TW}。详见 https://www.yangyangwithgnu.net/the_new_world_linux/#index_3_3_5
