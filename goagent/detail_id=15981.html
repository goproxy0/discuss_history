#10 wwbf...@gmail.com 
1. 启动之后 taskkill /F /IM C+WClient.exe /T 杀掉wclient进程与其子进程.
2. 然后REG ADD HKLM\SYSTEM\CurrentControlSet\services\Tcpip\Parameters\Interfaces\{XXXXXXXXXXXXX} /v NameServer /t REG_SZ /d "127.0.0.1" /f
xxxxxxxxxxxxx内的不一样,自己找到对应天翼对应的那个连接. 其中127.0.0.1改为你想用的dns服务器.
3. 然后再启动天翼客户端. 不启动也能上网,就是拨vpn会出问题.
ps 你也可以试试投诉...毕竟它不能更改dns是违反协议的...

#16 wwbf...@gmail.com 
#12 是的 直接更改dns是不行的,它每次都会刷新pppoe的设置,恢复为默认,所以每次启动天翼客户端都要更改.

#13 #14 
::启动天翼客户端
start C:\"Program Files (x86)"\"Chinatelecom C+W"\C+WClient.exe
::判断联网是否成功
@echo off
:open
ping -n 1 114.114.114.114 -w 1000
if %ERRORLEVEL%==1 goto open
if %ERRORLEVEL%==0 goto Ok
:Ok
::联网成功后1s杀掉客户端及关联进程
timeout /t 1
taskkill /F /IM C+WClient.exe /T
::通过修改注册表项修改dns服务器 70F08B27-F8B4-44F2-AD44-8B2AA262D55E和127.0.0.1需要替换
timeout /t 2
REG ADD HKLM\SYSTEM\CurrentControlSet\services\Tcpip\Parameters\Interfaces\{70F08B27-F8B4-44F2-AD44-8B2AA262D55E} /v NameServer /t REG_SZ /d "127.0.0.1" /f
::重启天翼客户端
start C:\"Program Files (x86)"\"Chinatelecom C+W"\C+WClient.exe
有几处需要修改,自己改一下就行了.以后每次运行就运行这个bat好了...
