first download Google App Engine SDK for Python  
https://code.google.com/appengine/downloads.html then find google_appengine_1.6.2.zip

move google_appengine_1.6.2.zip to the directory  where goagent has been unzipped.

in terminal, change the current directory to goagent, then enter google_appengine/appcfg.py update goagent/server/python/   to deploy goagent

#4 QQ510371...@gmail.com 
如果出现：urllib2.URLError: <urlopen error [Errno 32] Broken pipe>
则先用原来的版本联网，然后终端输入命令：
export https_proxy=127.0.0.1:8087
接着开始上传就不会出现那个错误了。
在server目录下运行：
python uploader.zip
我这样做了发现出现502,后来发现是golang的问题，上传python服务端就好了。
执行这个：
uploaddir=python python uploader.zip
一切ok了！

再整理一下我理解之后使用成功的办法：
1、到https://code.google.com/intl/zh-CN/appengine/downloads.html#Google_App_Engine_SDK_for_Python下载google_appengine_1.6.2.zip这个压缩包，然后解压缩到你之前解压缩goagent的目录下(假设为～/goagent)；
2、我看到我的～/goagent/server/python/app.yaml文件里，application:已经是我的ID了，所以不作更改；
3、在终端里进入～/goagent目录，执行google_appengine/appcfg.py update /server/python/，提示输入邮箱地址、密码，然后就OK了。
