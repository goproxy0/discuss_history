#4 zmot...@gmail.com 
出现这个提示就是python.py主进程停止响应了，打开任务管理器就会看到python.py这个进程的CPU占用保持在大约99%的样子。

解决方法很简单，手动把进程终止，然后重新启动goagent就行了。至于为什么Mac下会出现进程死锁，这就得技术大牛们研究了。

#12 xiaozhi6156@gmail.com 

用GoAgent的时候老是出现socket.error:[Errno 48] address already in use的错误,

6楼给的答案，发现可以

打开终端
sudo -s     (输入密码,获取权限)
lsof -i:xxxx    (xxxx是被占用的端口,得到占用端口的进程的PID)

例:COMMAND     PID     USER      FD      TYPE             DEVICE     SIZE/OFF         NODE                   NAME
       Python          919      root       3u     IPv4    0xffffff801703c880      0t0  TCP localhost:8087 (LISTEN)
kill xxx  (xxxx是占用端口的进程的PID)
ok,端口被释放.
