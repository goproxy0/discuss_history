Go Agent 配置：
[gae]
profile = google_hk

[google_hk]

#8 Legendar...@gmail.com 
我确定按你说的做了：

1. 首先我的goagent 2.1.3 是正常工作的
2. 在chrome中输入https://www.google.com/ncr，然后回车，打开这个主页
3. 然后进入chrome的设置，把默认搜索的网址改为https://www.google.com/search?q=%s，保存
4. 然后在浏览器中键入google.com，打开，然后“随便输入文字进行搜索 搜索结果右上角找到设置页面把搜索引擎语言改为中文”
5. 然后使用图片搜索，依然是重定向
