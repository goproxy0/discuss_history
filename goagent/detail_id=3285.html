我折腾了好久，终于在路由器上成功运行了 GoAgent，具体文章请参考：http://www.yonsm.net/post/645

经过筛选，Python 包减少到 3.5M以内（JFFS压缩，未压缩7M左右）。为了使 GoAgent能在大部分8M Flash上的路由器上运行，我对 proxy.py 做了少量修改：

1. 去掉了 love （因为love的的部分语句用到了 decode 函数，需要encoding支持）

2. 去掉了 logging （以便减少Python的 logging模块）

3. 合并 PAC 服务和代理服务为同一个端口，这样可以减少一个线程（路由器的CPU能力都比较差，越节约越好）。

如果可能，希望能考虑这种需求，即，尽量减少依赖（或者检查依赖项是否存在）并减少CPU/线程开销。

对于第三点，可以考虑做成如果PAC_PORT == LISTEN_PORT，则在同一个LocalProxyHandler中处理 PAC 请求。

#2 Yonsm...@gmail.com 
关于第三点，我现在的实现在你现有的功能的基础上做了一点点增强：假如PAC_PORT==LISTION_PORT则复用两个功能到一个端口上（你之前的版本如果这两个端口相同则会运行出错）。

我这个更改对于PC和路由器的代码都是一样的，只是配置文件中 pac 的端口更改一下即可选择是一个线程还是两个线程。看看我修改的部分：

第一部分，添加：
class LocalProxyAndPacHandler(LocalProxyHandler, LocalPacHandler):
	def do_GET(self):
		if self.path == '/'+common.PAC_FILE:
			LocalPacHandler.do_GET(self)
		else:
			LocalProxyHandler.do_METHOD(self)

第二部分，main中的最后部分更改为：
	if common.PAC_ENABLE and common.PAC_PORT != common.LISTEN_PORT:
		httpd = LocalProxyServer((common.PAC_IP,common.PAC_PORT),LocalPacHandler)
		thread.start_new_thread(httpd.serve_forever,())

	if common.PAC_ENABLE and common.PAC_PORT == common.LISTEN_PORT:
		httpd = LocalProxyServer((common.LISTEN_IP, common.LISTEN_PORT), LocalProxyAndPacHandler)
	else:
		httpd = LocalProxyServer((common.LISTEN_IP, common.LISTEN_PORT), LocalProxyHandler)
	httpd.serve_forever()

以上代码已经测试运行通过。
