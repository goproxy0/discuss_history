#5 Cickumqt 
Requirement of gevent since goagent 3.1 has been lifted to 1.0 at least, so you need to update the version existed on your system. However most of distros still carry 0.x for backwards compatibilities. After the testing of gevent 0.x series, we see that 0.x still can "work" on most of cases.

Hence I've write a patch and got merged in the upstream.

Please use eithor:

1. Use pip to update the package manually(Note this may break the system package consistency if you install the library from apt-get/yum/urpmi). If you just want to install this module from distribution repo, please move to selection2:

2. Checkout from github manually and use that version. A warning will be throwed when detecting the old version gevent.

3. Roll back to 3.0.9
