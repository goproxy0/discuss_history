#36 radio....@gmail.com 
扫到的gvs全部在ip_tmpno.txt文件里面，checkgoogleIP的开发者说这个文件存放的是不能用于GAE的，不知带楼主是否弄错了？
GAE的IP应该在ip_tmpok.txt里面，我的全部只有GWS，所以更改了ip_gvs.bat为筛选GWS IP：
iplist = []
with open('ip_tmpok.txt') as ip_tmpno:
    for x in ip_tmpno:
        sline = x.strip().split(' ')
        if len(sline) == 4 and sline[3] == 'gws':
            iplist.append(sline)
with open('ip_tmpno_ok.txt', 'w') as ip_tmpno_ok:
    iplist.sort(key=lambda x: int(x[1]))
    out = '|'.join(x[0] for x in iplist if int(x[1]) < 1000)
    # print(out)
    # print(iplist)
    ip_tmpno_ok.write(out)
