@clowwindy's ChinaDNS can "Fix weird things with DNS in China". It's "A DNS forwarder that ignores incorrect(you know it) responses".
https://pypi.python.org/pypi/chinadns
Windows
http://sourceforge.net/projects/chinadns/files/dist/ ( rar file )
OpenWRT
http://sourceforge.net/projects/chinadns/files/dist/ ( ipk file )
Archlinux
yaourt -S chinadns ( could be voted to community repo )
Ubuntu ?
( I don't know. )

Usage:
Windows
Just run the exe file. 
If want to hide the running window, save http://www.robvanderwoude.com/files/runnhide_vbs.txt as runnhide.vbs, then make it a shortcut with this parameter x:\path\to\chinadns\dnsrelay.exe
OpenWRT
opkg install ChinaDNS-C_1.x.x_ar71xx.ipk
/etc/init.d/chinadns start
Archlinux
sudo systemctl start chinadns
Ubuntu ?
( I don't know. )

Then set your DNS to 127.0.0.1, and ONLY to 127.0.0.1, just this one record.

Good way: you can build binary ( Windows exe file ) by yourself.
Step 1
1. download & install https://www.python.org/download/releases/2.7.8/

2. download & install
http://sourceforge.net/projects/py2exe/files/py2exe/0.6.9/py2exe-0.6.9.win32-py2.7.exe/download
3. restart computer
Step 2
1. download & unzip https://github.com/clowwindy/ChinaDNS/archive/master.zip
2. download & unzip https://github.com/clowwindy/shadowsocks/archive/master.zip
( make sure this two unzipped folders have a same father folder )
Step 3
1. run py2exe_make.bat in ChinaDNS folder to build
2. run dnsrelay.exe in ChinaDNS\dist folder to check if it works
Good luck !

#25 2007st...@gmail.com 
我来总结一下 ChinaDNS 在视窗操作系统中的操作步骤：

1.  下载地址 http://sourceforge.net/projects/chinadns/files/dist/ChinaDNS-win32-0.1.11.rar/download

2.  解压缩后，将 http://www.robvanderwoude.com/files/runnhide_vbs.txt 中的内容保存为runnhide.vbs，在启动文件夹中创建runnhide.vbs的快捷方式，将目标指向dnsrelay.exe所在的路径。样例如下："X:\xxx\ChinaDNS-win32-0.1.11\dnsrelay.exe"

3.  将网卡中的“首选DNS服务器”设置为 127.0.0.1 下方的备用DNS服务器空着。

4.  将 ...\GoAgent\local\proxy.ini中的[dns]设置为
    [dns]
    enable = 0

5.  重启计算机，启动goagent。畅游自由网络吧！
