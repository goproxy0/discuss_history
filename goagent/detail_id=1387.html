#14 xiongwan...@gmail.com 
read the Readme file when you double click the GoAgentMac.dmg to install it. See you must Change the GoAgentPath in GoAgentMac.app/Contents/Info.plist eg:/Users/hewigovens/Downloads/goagent/local/proxy.py. And you should never change the path directly in the install step because there may occur #12 says this volume is read only. All you should do is as follows:
1. drag the GoAgentMac to Applications.
2. go to Application and right click on the GoAgentMac to show package content.
3. find GoAgentMac.app/Contents/Info.plist and change the GoAgentPath to where you place the goagent`s proxy.py. Eg:/Users/hewigovens/Downloads/goagent/local/proxy.py
