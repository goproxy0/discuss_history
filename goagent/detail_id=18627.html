Reported by playne...@gmail.com, Nov 15, 2014 
我看最新版本的软件关闭了headfirst选项，我配置文件使用的是自定义的IP，如果关闭配置文件中的headfirst项，观看视频效果会打折，没有开启效果好，建议开启headfirst＝1和validate=1和obfuscate = 1三个选项。

headfirst = 0
keepalive = 0
obfuscate = 0

options = rc4
regions = cn
