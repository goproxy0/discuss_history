#2 lvp0...@gmail.com 
遇到相同的问题，goagent所在目录含中文，certfile变量被转为unicode string，ascii codec再编码失败。我的解决方法：
#line 302
--   return os.system('cp "%s" "%s" && update-ca-certificates' % (certfile, new_certfile))
++ return os.system('cp "%s" "%s" && update-ca-certificates' % (certfile, new_certfile.encode('utf-8')))

------------
ubuntu 12.04.3 64位
系统编码utf-8
python内建编码 ascii
