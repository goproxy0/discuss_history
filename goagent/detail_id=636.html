Issue 636: 关于代理密码的使用问题。

Reported by huguangz...@gmail.com, Aug 5, 2011 
由于流量问题，所以在gae上建立的了多个goagent应用，以前建立的服务器1没有有密码，新建的两个服务器分别设置了不同的密码，问题在于 goagent的win客户端下 porxy.ini文件可以设置使用多个服务器，但是无法分别设置密码。
使用如下格式设置，则服务器2工作，服务器3和1都在限制。
appid = 服务器2|服务器3|服务器1
password = 密码2
如下设置提示密码错误
appid = 服务器2|服务器3|服务器1
password = 密码2|密码3|空
希望新客户端能够提供对多密码的支持
