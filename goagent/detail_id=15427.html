最好不要使用goagentx，不稳定。

你下载goagent的源码压缩包（直接用safari下载，会自动解压缩的，记住将被系统自动删除的goagent的压缩包放到随便哪个你想放的文件夹里面），然后使用如下命令移动goagent文件夹并进行配置。

safari下载的文件一般都是在下载文件夹里面的！！！

sudo mv ~/Download/goagent-文件夹名称 ~/goagent

然后打开使用你用户名的哪个文件夹，就可以看到goagent文件夹了，进去配置goagent

然后在/goagent/local文件夹下面新建一个文件：start.command，用文本编辑打开这个文件，输入下面两行内容：

#!/usr/bin/env sh
sudo ${0%/*}/goagent-osx.command

关闭文件

然后给这个文件添加权限：

sudo chown -R 你的用户名 ~/goagent/local/start.command 

sudo chmod +x ~/goagent/local/start.command

然后将这个文件加入开机启动就可以了！！！

1、系统偏好设置--->用户与群组--->登录项--->点“＋”号，选择start.command 文件

2、打开钥匙串选择：登录--->证书--->点“＋”号，将~/goagent/local/文件夹下面的CA.crt文件加入，显示的文件名应该是GoAgent CA，双击它--->点开“信任”选项--->选择“总是信任”。

3、同样的方法将证书文件添加到：系统--->证书里面。

4、打开你的网络--->高级--->代理，勾选第一和第二两个自动开头的选项，在自动代理配置里面输入pac文件的地址，一般都是：http://127.0.0.1:8086/proxy.pac，最后保存，应用就全部完成goaagent的配置了。

接着就安心无干扰的使用吧！！！

补充几点：

在Mac系统下使用goagent，你应该会遇到goagent无法启动的问题，使用一下命令安装软件就能全部解决了：

1、install Xcode（这个装不装看个人需要，可在App Store中安装）

2、install homebrew

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

3、brew install libevent

4、sudo easy_install pip

5、sudo pip install cython

6、sudo pip install greenlet

7、sudo pip install gevent

9、sudo pip install pycrypto
