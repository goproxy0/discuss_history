对新旧版本发表一下观点，GoAgent 绝对不会因为新的特性而放弃速度考虑。
一个底限就是: 如果某个特性是关闭的，那么和它相关代码绝对不会影响速度。
因此，请不要有某某特性拖慢速度的顾虑。

#10 phus.lu 
pagespeed 使用的是 g.cn 服务器。架构上大概是这样

proxy.py -> google_cn的pagespeed服务器 -> google_hk的GAE服务器 -> 目标网站。

好处就是如果你连接 g.cn 的速度稳定快速，那么效果很好，但并不是所有地区都这样。
坏处是，pagespeed 只能携带 16K 的数据，所以想用它发图片/视频的话，就不能了。

#23 ben...@bensontech.net 
回覆20樓:
只要分別設定不同埠位址就可以共用兩個版本
例如在3.1.18參數檔proxy.ini
[listen]
ip = 127.0.0.1
port = 8088
visible = 1
debuginfo = 0
那3.1.19參數檔proxy就設成
[listen]
ip = 127.0.0.1
port = 9088
visible = 1
debuginfo = 0
如果有使用PAC或是PHP(PAAS)也一樣把port位址分別設定不同位址
