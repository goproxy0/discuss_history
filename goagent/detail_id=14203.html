#89 qgap...@gmail.com 
试了下，0608-2中还是存在测试低延时的时候如31#所述，无法获得结果的问题
看了下，似乎与TcpPingCount的加减有关，我在程序里加了几句，好像可以解决此问题，供lz参考。

IP.CS:

class IP中，增加：private static object countlock = new object();
public void TcpPing()：
            if (Ok80 == Unknow)
            {
                lock (countlock)
                {
                    TcpPingCount++;
                }

            if (Ok443 == Unknow)
            {
                lock (countlock)
                {
                    TcpPingCount++;
                }

EndTcpPing80：
            lock (countlock)
            {
                if (Ok80 == Unknow)
                {
                    TcpPingCount--;
                    Ok80 = okstr;
                }
            }
EndTcpPing443：
            lock (countlock)
            {
                if (Ok443 == Unknow)
                {
                    TcpPingCount--;
                    Ok443 = okstr;
                }
            }
