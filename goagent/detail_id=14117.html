Issue 14117: ipv6隧道接入用户科学上网方法
6 people starred this issue and may be notified of changes. Back to list
Status:  New
Owner:  ----
Type-Defect
Priority-Medium


 
Reported by anesli...@gmail.com, Jun 5, 2014 
有许多童鞋使用ipv6隧道技术接入ipv6网络，配合ipv6 hosts访问google等提供ipv6的网站，但是目前ipv6 hosts停止更新，许多地址失效，导致网络访问不畅。这里提供一个方法，让你的电脑优先解析和使用ipv6地址科学上网，目前gfw还没有干扰ipv6。
1.ipv6隧道技术有多重，包括isatap，6to4，teredo等。首先确认你的ipv6隧道配置成功，已成功连接至ipv6网络。测试网址：www.test-ipv6.com。（如果还没有配置ipv6隧道的童鞋想尝一下鲜，可以自己google一下ipv6隧道的配置方法，网上教程很多。）
提高ipv6隧道优先权：（cmd命令窗口依次输入如下命令）
netsh int ipv6 set prefix ::1/128 50 0
netsh int ipv6 set prefix ::/0 40 1
netsh int ipv6 set prefix 2002::/16 30 1
netsh int ipv6 set prefix ::/96 20 3
netsh int ipv6 set prefix ::ffff:0:0/96 10 4
netsh int ipv6 set prefix 2001::/32 5 1

经过以上设置你的电脑已经可以优先访问ipv6网络。
2.曾经配置过ipv6 hsots的童鞋请删除hosts中ipv6地址。

3.由于ipv6隧道不支持定义ipv6 dns，任然使用ipv4 dns解析，由于gfw干扰，经常导致解析不正确，所以需要用的dnscrypt这个软件进行加密dns解析。windows使用dnscrypt请看这里http://www.douban.com/note/352893311/。配置完成后，你是不是发现google，youtube，facebook等支持ivp6的网站可以不用代理访问了。本人测试6m联通宽带youtube 1080p无压力缓冲嗖嗖的。

申明：使用本方法你的通信数据未进行加密，你的行为同样受到监控，请勿访问反对、色情、暴力网站和发表不适言论。^_^
