Issue 14686: 【怎样给 goagent 换 ip】（给小白们写的，高手不用看）

Reported by coocle.m...@gmail.com, Jun 15, 2014 
goagent 有一个文件叫 proxy.ini，其中有两行字，分别是：

google_cn = www.google.cn|www.g.cn|...
google_hk = www.google.com|mail.google.com|...

等号后面的网址（用竖线|隔开），由于 GFW 的原因，越来越不好用，导致屏幕上经常出现黄字红字，速度很慢，有时甚至无法使用，此时就需要用新 ip 替换这些网址。

怎么替换呢？一种方法是使用本坛网友开发的专门软件 gogo tester，自己找新 ip，自己测试新 ip，然后用测试通过的新 ip 替换这些网址，例如；

google_cn = 61.25.74.13|61.25.74.15|61.25.74.17|...
google_hk = 61.25.74.13|61.25.74.15|61.25.74.17|...

但这种方法对小白们有一定难度，所以推荐第二种方法，那就是去下列网站：

http://qooqle.yuku.com

那里有现成的新 ip，拿过来就能用。挑选新 ip 时，请务必分散挑选，比如韩国选几个，丹麦选几个，加拿大选几个，千万不要集中在一个国家。

大体上就是这个样子啦，祝您成功。
