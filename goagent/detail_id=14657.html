Reported by ikonigl...@gmail.com, Jun 14, 2014 
    我觉得Ping IP还是有点小麻烦的,虽然不是很占用多少时间就能得到几个可用IP.但是我觉得还是在自己掌控范围内比较靠谱,所以还是觉得Goagent PHP版的还算可以.当然还是要分目的性,我一般都是查资料之类的,看视频真的无爱也没时间.所以在这小述一下PHP版吧,虽然网上教程多的数不胜数.
    1.如果您是高手,那么以下步骤完全可以忽略,在下仅仅提供一种方法.如果您要看视频,可以Ctrl+w说再见了
    2.找一个PHP服务器,现在PHP服务器真的是一抓一大把.个人建议既然当DL,那么空间大小可以不用在意,主要是看流量以及价格.其实有很多无限流量或者大流量的小容量的免费空间.我觉得大家应该会找到(在下有一个国内的VPS略微有点蛋疼).
    3.server文件夹里面有index.php文件,修改密码,例如:$__password__ ='123123';然后上传到服务器
    4.在proxy.ini里面添加
         [php]
              enable = 1
              password = 123123 ;改成你的Password,与index.php的对应
              validate = 0
              crlf = 0
              listen = 127.0.0.1:8088
              fetchserver = http://mysite.com/index.php;文件的Url地址,自行定义
    5.打开Goagent,在浏览器内添加并切换到8088端口
Ok,差不多就可以了.
