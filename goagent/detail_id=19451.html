Issue 19451: Mac's GoAgent 一个解放办法
1 person starred this issue and may be notified of changes. Back to list
Status:  New
Owner:  ----
Type-Defect
Priority-Medium


 
Reported by seo...@gmail.com, Jan 11, 2015 
元旦前后，Mac 下的 GoAgent 就挂了，动不动黄字红字，换了一批IP、最新版也无法解决。上这里一看，大家都说是 DNS 被污染，SSL 端口被封啥的，这些偶都不太懂。我只想能看想看的东西。

说下付费，付费如VPN啥的，能稳定简便也是可以接受的，如果是手机上。但PC上，我觉得不是这样的。有个好东西原来是免费的开放的，后来被禁或啥的用不了，现在换一个或者付费或者可以解决。但这已经不是付费与否的问题了；这个好东西已经是我日常工作生活的一部分，现在是好东西怎么能重新用起来的问题。

——我的思路就是开虚拟机来跑 GoAgent。因为我们都知道，现在同样网络同样配置，Win OK，Mac 是万万不 OK 的，甚至连谷歌首页都打不开。或许有点麻烦，不过工作需要，我本来就需要开虚拟机哈哈：）

操作系统版本：OSX 10.10.1
浏览器版本：Chrome 39 x64
虚拟机：VMware Fusion 7 with Win7
GoAgent 版本：3.1.22


1、来看看 DNS 的问题

A、我能想到最简单的办法：

直接修改为 谷歌、百度、阿里 等的公共 DNS。经测试，百度较快，谷歌算稳定，阿里经常断。

谷歌DNS：8.8.8.8, 8.8.4.4
百度DNS：180.76.76.76, 114.114.114.114
阿里DNS：223.5.5.5, 223.6.6.6

——为什么百度这么臭名昭著还用它的服务呢。很简单，因为相对来说还算好用。这里我不排除它背后会偷偷做什么手脚，谁知道，反正我自己又做不出一个公共 DNS 来。不过有一点，我不会点任何百度的广告，也不会用任何百度的付费服务就是。

B、如果上面的步骤还出现黄字红字：

可以考虑用 DNSCrypt (http://www.williamlong.info/archives/3890.html)，Win7 直接下载安装，完了开机会自动运行，一般绿色表示生效。

——测试办法：用谷歌图片随便搜，然后下拉。有问题的话很快就会黄字红字然后出错。


2、IP 的问题

A、自己寻找几十个可用 IP 还是要的。其实大部分 IP 都没问题，只是被 DNS 糟蹋了。

B、Win7 运行 GoAgent，然后将子机的 IP 设置到 Mac 上 Chrome 的 SwitchySharp。


——如今我 Mac 跑的妥妥的。Youtube 啥的都没压力。
