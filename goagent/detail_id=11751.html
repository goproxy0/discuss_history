先安装pip，不知道怎么回事一直easy_install不成功
之后sudo pip install greenlet
之后sudo pip install cython git+git://github.com/surfly/gevent.git#egg=gevent
如果提示缺失crypto，还是pip安装之sudo pip install pycrypto
我最后居然一直提示缺失openSSL，后来发现似乎是版本问题，于是sudo pip install --upgrade openssl
整个过程期间一直各种warning触目惊心，总算最后没出什么问题
