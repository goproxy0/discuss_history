#16 shuangzh...@gmail.com 
#15
从vista时代就已经默认支持ipv6，至于从IPv4过渡到IPv6，有6to4，isatap，teredo以及tunnelbroker服务。前面2种方法都要求你的电脑本机必须具有公网IP，没有公网IP前2种方法是不会连得上，除非你在高校的校园网。如果你的网络经过路由器的，一样不能使用。teredo对网络类型没有要求，但是teredo因为在IPv6时代被认为是多余的连接方式，所以即便能够解析到网站的IPv6地址也不会默认连接IPv6，必须做hosts才可以使用teredo隧道连接IPV6网站。请先自行补充IPV4过渡到IPV6的知识吧。

#36 shuangzh...@gmail.com 
再进一步说，6to4地址形式为
2002:你的IPV4地址的16进制形式:你的IPV4地址的16进制形式
6to4的relay服务器基本都是192.88.99.1
2002:c508:6301::c508:6301网关地址里面的c5086301就是192.88.99.1的16进制写法

微软默认使用自家的6to4服务是为了更快的普及ipv6的使用，不是电信分配的原生IPV6地址。fe80开头的IPV6地址就像是IPV4里面的192.168.*.*一样是私有的，并不是用于公网访问

有公网IP地址的我更建议使用isatap来连接IPV6
cmd执行
netsh int ipv6 isatap set router 222.22.32.161
netsh int ipv6 isatap set state en
即可使用中国教育网的IPV6网络。但是从命令中你是不是看到仍然使用的是IPV4的服务器地址？仍然不是Native原生的IPV6地址。你的访问仍然是通过222.22.32.161这个IPV4服务器进行通信的
