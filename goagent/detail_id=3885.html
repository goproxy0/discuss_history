#10 totol...@eml.cc 
我在用西海岸一个名机房的共享空间，表示可以正常使用，无限空间无限流量，速度杠杠的，也很便宜。
免费空间禁用很多函数，也不稳定，找一找吧，还是有能用的

我分享一点避免gfw，避免被免费空间发现的经验
你尽量勤快点更新php的服务端，
php端设置一个密码

function get() {
    header('Location: https://www.google.com/');
}
找到这句，中间这行最好换个网址或者删掉一行

最后，找zend或者ionCube，把代码丫一丫
http://www.ioncube.com/online_encoder.php
注意先找个探针：yahei就行，对上空间的php版本和cpu型号，别压错了
