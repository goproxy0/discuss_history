#15 PenkZ...@gmail.com 
可以尝试这个方法，我试了 还行 就是用gae上传  

在使用ubuntu安装goagent时，在终端用命令行输入python uploader.zip update ./ 按回车上传时总是报错显示：

urllib2.URLError: <urlopen error [Errno 32] Broken pipe>

解决办法：

linux下载好appengine后，
（download Google App Engine SDK for Python https://code.google.com/appengine/downloads.html，unzipped）

执行下面两步即可，不用照主页上面的说明生搬硬套
1.修改ID(通用方法)：goagent/server/python/app.yaml
2.用appengine上传：python google_appengine/appcfg.py update goagent/server/python/

（出处：http://www.zhixing123.cn/ubuntu/24437.html）
