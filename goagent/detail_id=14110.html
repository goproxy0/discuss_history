#8 arbiterazzvx 
回#6 ：
关于这个软件需要解释一点：ping只是一个参考值，如果ping过大，那么这个ip就算可用也不好用，所以ping过大的ip会被丢弃。
还有，这个测试ip是否可用是通过 GoAgent 测试的，他会修改 proxy.ini 文件，重启 GoAgent ，然后通过 GoAgent 代理访问网址，如https:\\www.google.com 。
如果访问正常，返回也必须返回 200 状态码，那么说明这个 ip 是正常的，可用的。如果无法访问或者返回其它状态码，那么这个 ip 就有问题。
Jun 5, 2014
#9 arbiterazzvx 
另外，软件在测试 proxy.ini 会被修改成：
[iplist]
google_cn = 测试IP
google_hk = 测试IP
google_talk = 测试IP
google_ipv6 = www.google.com|www.google.com.hk //这个不修改，对ipv4无用

[ipv4/hosts]
mtalk.google.com = google_hk
