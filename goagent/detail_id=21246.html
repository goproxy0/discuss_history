Issue 21246: Koding 主机不更换IP的使用方法

Reported by xyuanmu, Jun 1, 2015 

因为云主机申请后现在使用的比较少，之前的经验不足，现在给大家补充点 Koding 主机的使用方法，已经知道的就飘过啦。。飘过。。。

之前在一篇“Cloud 9，Koding, Nitrous 几大IDE云比较”的帖子（Issue 21063）中说道，Koding 的主机需要使用ip作为本地 gost 客户端的目标地址，可是 Koding 每次重启后ip都会有所变化，这里给大家补充下 Koding 的固定地址。

在把鼠标移到你新建的应用上，会在右边出现一个图标，里面有三个点的圆圈，点击这个图标，复制 Assigned URL 的地址，并在其后加上 :8080 就是 gost 的目标地址而且是固定不变的。

这个是在下的：http://uykk87909ddf.yuanmu.koding.io:8080/
本地客户端就这样填写：gost -L :8899 -S uykk87909ddf.yuanmu.koding.io:8080 -p 密码 -m 加密方式 -ws

另外提个醒，如果觉得默认的域名太长不好记，可以切换到 Domains 标签，里面有一个默认带有用户名的二级域名，使用这个二级域名也可以，就是：
http://yuanmu.koding.io:8080/
gost -L :8899 -S yuanmu.koding.io:8080 -p 密码 -m 加密方式 -ws
