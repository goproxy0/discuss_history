此页只提供程序及升级，请大家有问题或有留言在下面连接发表，别把下面的楼道都占了。

https://code.google.com/p/goagent/issues/detail?id=14644

本软体可以在Google，20多万个ip池里寻找有效的ip，供goagent使用。全部扫完可以找到2000个左右的ip, 程序具有自动测试ping，并会把延时短的ip放前面，程序使用请看附件包里的README。

使用本软体的协议是GPLV2.0及以上。

注：对苹果用户，没有机器，无法评测。

版本更新：V1.3，修复若干Bug。优化内存占用和降低CPU使用率。请用新版本。

对于老机器，请自行修改find_host_port.py中的

thread_num=100

建议修改成10, 可以参考：thread_num=10时，在P4，256MB内存的机器上调试可行（内核占用92MB）。

修改后，再根据README运行，谢谢。

建议大家运行find_ip.sh时，找到了几百个IP后，就Ctrl+C。再运行./deal_ip.sh，若等find_ip.sh自行完毕，我这边测试显示，时间很长，当然也能找到2000个左右的IP。而./deal_ip.sh的原理是，每5秒测试10个IP的ping时间，再把延时短的放前面。时间会比较长。

#29 Monistra...@gmail.com 
#27, I tried to install freegate on ubuntu14.04 or ubuntu 14.10, it works well.

On Debain, you can try but not tested,

First, sudo apt-get install wine
Then, download freegate at http://www.dongtaiwang.com
Last, copy windows's mfc42.dll(in C:\Windows\system32) to the folder contain freegate.

Double click fg742p.exe, wait for some time, it will be OK.

More:
    Create shortcuts for freegate:

    Please save below to freegate.desktop

   --------------------file start-------------------------
[Desktop Entry]
Version=7.42
Name=Freegate
Comment=Free
Exec=wine /opt/freegate/freegate.exe
Icon=/opt/freegate/freegate.png
Terminal=false
Type=Application
Categories=Wine;

------------file end--------------------

Modify /opt/freegate/freegate.exe  and /opt/freegate/freegate.png to yours(you can Google a image for freegate)

After that, cp this file to ~/.local/share/applications/
