#9 saphirza...@gmail.com 
以目前首页上可下载的 3.1.22-30-gd4c7dda 版本为例

1. 删除自带的证书(CA.crt和certs目录)
   自带的证书iOS可导入,但Android不可导入
   不用自己手工创建证书, goagent重新运行后自建的证书即可同时导入Android和iOS

2. 修改proxy.ini文件
   也可添加在proxy.user.ini文件中
   [profile/ipv4]下指定 .googlevideo.com = withgae

3. 禁止autorange
   需要注释掉proxy.py的444,445,446行, 如下
            # if response.status == 206:
            #     fetchservers = ['%s://%s.appspot.com%s' % (self.mode, x, self.path) for x in self.appids]
            #     return RangeFetch(handler, self, response, fetchservers).fetch()

上述方法, 在Android 4.2.2, Android 4.3, Android 4.4.2, iOS 7.1.2 上验证, youtube应用都能正常执行

因autorange被禁止了, 会导致整体速度不及pc上稳定, 有时会有卡顿, 如出现这种现象可适当降低视频的分辨率
可在pc上运行两个goagent, 侦听在不同的端口, 一个不修改用作pc代理, 一个如上修改用作移动设备代理
